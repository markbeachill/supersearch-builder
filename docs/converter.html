Search link converter

Turn a normal search-results link into a reusable Super Search template. The converter finds the words you searched for inside the link and replaces them with a %s token — so the same engine can be searched for anything later.

Once you have a template you are happy with, open the Builder and paste it into a section, or use the builder's built-in "Add engine from a search link" panel which runs this same converter for you.

Why links look so different

Different sites put your query in different places. The converter handles the common shapes automatically:

?q=%s            Google, DuckDuckGo, Reddit, CORE
?Query=%s        JSTOR
?search=%s       Wikipedia, Wikimedia
?term=%s         PubMed
?search_query=%s YouTube
?query=%s        Project Gutenberg, OpenAIRE
/search/%s       some sites put the query in the path
climate+migration  or  climate%20migration   both encodings are recognised
