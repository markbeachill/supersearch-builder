For research

One query. Many searches.

Build a standalone Super Search page: type a query once and open it across academic databases, repositories, news, media and reference sources — each with its own search pattern handled for you. No account, no extension, works on mobile, and the page is a single HTML file you own.

Start with the research template Try the link converter
Handles varied patternsDifferent engines put your query in different places — ?q=, ?Query=, ?term=, in the path. The builder stores each as a template with a %s token.
Add any engineRun a real search, copy the link, paste it into the converter. It finds your phrase and turns the link into a reusable search template.
Standalone outputExport a self-contained HTML file. Keep it local as your homepage, or publish it on GitHub Pages and use it anywhere.
