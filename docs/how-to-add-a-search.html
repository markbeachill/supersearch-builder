How to add a search engine

Every engine in a Super Search page is a search template: a web address with a %s token marking where your query goes. You do not have to write these by hand — capture a real search and let the converter do it.

The three steps

Search for a memorable phrase
On the site you want to add — a library catalogue, a database, a journal, a news site — run a normal search for a short, distinctive phrase such as climate migration. Avoid single common words, which can appear elsewhere in the address.
Copy the results-page address
When the results appear, copy the full address from your browser's address bar. It will contain your phrase, for example …doSearch?Query=climate+migration.
Convert and add
Paste the address into the link converter (or the builder's "Add engine from a search link" panel), tell it the phrase you searched, and it produces a template like …doSearch?Query=%s ready to drop into a section.

What the converter recognises

Your phrase might be encoded in the address in different ways. The converter checks the common forms automatically, so you do not need to know which one a site uses:

climate+migration      spaces written as +
climate%20migration    spaces written as %20
in the query string    …?q=climate+migration   →  …?q=%s
in the path            …/search/climate%20migration  →  …/search/%s
with extra settings    …?q=climate+migration&sort=date  →  …?q=%s&sort=date
If the converter cannot find your phrase: some sites send searches as hidden form data rather than putting them in the address, so the results address does not contain your words. In that case the search cannot be turned into a simple template. Try the site's own search box from your Super Search page instead, or look for an "advanced search" form that puts terms in the address.

Worked example

Searching JSTOR for climate migration gives an address like:

https://www.jstor.org/action/doSearch?Query=climate+migration

The converter finds climate+migration and returns the template:

https://www.jstor.org/action/doSearch?Query=%s

Added to your page, typing any query into the search box now searches JSTOR for it.
