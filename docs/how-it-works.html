How it works

One query, many searches

A Super Search page has a single search box. You type a query once, press Search, and a set of links appears — each one runs your query on a different site. Try it right here:

Press Enter or click Search. Nothing is sent anywhere until you click one of the links that appears.

Why the links look different

Each link is just your query dropped into that site's own search address. Most sites take the query straight, like ?q=your+query. Some have poor search of their own, so the link uses a Google site search instead — that is the red-edged one above. The builder handles both for you.
