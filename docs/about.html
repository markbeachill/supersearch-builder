About

Super Search Builder makes standalone "super search" pages aimed at research. A super search takes a single query and fans it out into many searches — one per engine, database or site.

It is a static GitHub Pages site with no backend, no account and no external libraries. Everything happens in your browser. The page you export is a single HTML file that does not depend on this website. It is a sibling project to the Personal Start Page Creator and shares its design and config approach. The essential difference is that each link is a search template with a %s token, and one shared query box drives them all.
